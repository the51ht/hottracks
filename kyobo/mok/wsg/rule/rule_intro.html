화면상으로 볼 수 없는 내용의 확인

  • 링크/버튼요소에는 반드시 텍스트를 사용할 것
  • 다음과 같은 경우 구분선을 CSS로 처리하지 말고 구분선 기호( | )를 사용
  • 그밖에 잘못된 내용이 들어갈 수 있는 속성 확인
    • alt=""
    • title=""
    • <caption></caption>
    • for="" (id와 연결)
    • 그 밖에 숨김 처리한 대체 콘텐츠

웹접근성

  • 상품 이미지의 alt 값은 상품의 이름, 저자 사진의 alt 값은 저자의 이름으로 삽입
  • body 상단에 본문 바로가기 링크 제공
  • 탭 메뉴와 내용이 분리된 경우 따로 제목이 없다면 숨김처리 된 제목을 사용
  • 팝업창 열기 예시
  • 입력요소에는 반드시 label 연결 또는 title 값을 지정할 것

    복사하기/붙여넣기 시 id, for 값의 연결에 유의

  • 분리된 입력폼은 각각 title을 지정
  • 적절한 label 텍스트 또는 title을 사용할 것
  • 사용자 서식 체크박스를 사용할 경우 키보드 접근성 고려

    참고로 display: none; 인 경우 키보드로 접근이 불가능함

  • 키보드 탭키 이동으로 접근이 불가능한 내용이 없어야 함
  • 링크, 버튼 또는 입력폼에 focus 됐을 때 시각적으로 변화를 인지할 수 있어야 함
  • 자동롤링되는 배너는 마우스 오버 또는 키보드 포커스 상태에서 멈춤 처리

메뉴 관련 유의사항

상/하단에 고정된 메뉴 사용시 유의사항

  • 해시(#)링크로 이동한 내용이 화면에 고정된 메뉴로 가려지고 있지 않은지 확인
  • 키보드 탭이동으로 포커싱 된 입력폼이 화면에 고정된 메뉴로 가려지고 있지 않은지 확인
  • 스크롤 도중에 특정 위치에서 고정되는 메뉴는 움직임이 끊기지 않고 자연스럽도록 처리바람

GNB/LNB 유의사항

  • GNB : 좌우 스크롤되는 메뉴

    우측에 있어서 바로 안보이는 메뉴가 해당 페이지로 이동하여 ON된 경우 해당 메뉴가 보일 수 있는 위치로 스크롤되어야 함

  • LNB : 접혀있는 서브메뉴

    접혀있어서 바로 안보이는 메뉴가 해당 페이지로 이동하여 ON된 경우 해당 메뉴가 열린상태로 보여야 함

공통 레이어 팝업 구현시 유의사항

  • 레이어 팝업이 화면 가운데 위치하고 높이가 가변일 경우 다음의 상황을 고려
    • 내용이 길어서 스크롤이 필요할 경우
    • 탭 메뉴 또는 AJAX 처리로 나중에 내용 길이가 바뀌는 경우
    • 창이 리사이즈, 스크롤 되는 경우

    ※ 레이어 팝업의 위치와 크기 변경을 위한 스크립트 사용은 지양, CSS로 구현 권장

  • 레이어 팝업 열기 링크와 레이어 팝업은 ID로 연결
  • 레이어 팝업을 닫았을 때 해당 팝업의 열기 버튼으로 포커스가 이동할 것 (웹접근성)
  • 레이어 팝업을 열기 전과 닫았을 때 스크롤 위치가 같을 것

XHTML 문법

요구사항에 따라 DTD는 HTML5로 선언하나 코드의 통일성을 위해 XHTML의 엄격한 문법을 일부 준수하도록 하며
기준은 W3C Markup validator에서 Error나 Warning이 나지 않도록 한다.

  • 모든 요소와 속성은 소문자로 작성한다.
  • 속성의 값은 따옴표로 감싸준다.
  • 종료 태그가 있는 경우 닫기 태그를 반드시 선언한다.
  • 종료 태그가 없는 태그도 종료되어야 한다.
  • 중첩에 오류가 없어야 한다.

속성 따옴표

  • html 속성은 큰따옴표(""), CSS와 자바스크립트 속성은 작은따옴표(')를 사용한다.
  • 스크립트에서 HTML을 제어할 때 이스케이프 문자(\)의 사용을 줄일 수 있다.

코드 들여쓰기

  • 마크업의 중첩이 깊어질 때마다 자식 요소는 1탭 들여 쓰고, 1탭의 크기는 공백 4칸으로 설정한다.
  • 들여쓰기를 위한 공백문자는 Tab으로 통일하여 사용한다.
  • 모든 줄의 끝에는 공백문자(Space or Tab)를 포함하지 않는다.
  • ※ 다음의 경우에는 들여쓰기를 하지 않는다.
    • html의 자식 요소인 head, body
    • head의 자식 요소인 meta, link, script
    • body의 자식 첫 번째 요소

  • ※ 2 × 3 × 4 × 5 = 120
  • ※ 120의 약수: 1, 2, 3, 4, 5, 6, 8, 10, 12, 15, 20, 24, 30, 40, 60, 120
  • 360
  • 480
  • 720 (768: 태블릿)
  • 840
  • 960 (1024: PC 저해상도)
  • 1080
  • 1200 (1280: PC 일반)
  • 1320
  • 1440 (1920: PC 고해상도)

소수점 사용

  • px 단위 수치는 정수 값 사용

    12.5px (X) / 13px (O)

  • %, em 단위에서 소수점 사용 시 최대 2자리 사용

    3.1415926535% (X) / 3.14% (O)

  • 보다 정확한 디자인 적용을 위해 텍스트 자간(letter-spacing)의 값은 최대 3자리까지 사용
    • 포토샵 자간이 -25 일 경우: -25 / 1000 = -0.025em
    • 포토샵 자간이 10 일 경우: 10 / 1000 = 0.01em
