Button 공통규칙
  • 버튼의 중요도에 따라 오른쪽부터 왼쪽으로 배치되도록 해야한다.

  • [ 태그 구분 ] 버튼 기능에 따라 사용 필요 * 모든 버튼 타입 해당

    • a 태그 - 링크성 버튼일 경우 사용

      ┗ 비활성화시 : .disabled클래스 추가

    • button 태그 - event function 발생되는 버튼일 경우 사용

      ┗ 비활성화시 : disabled속성 추가

  • 버튼 class는 반드시 size 클래스 1개 + color 클래스 1개의 조합으로 작업한다.

  • [ size 클래스 추가 지정 ] 버튼의 size는 height 기준으로 구분한다.

    • .btn_lg : height: 56px
    • .btn_md : height: 50px
    • .btn_sm : height: 44px
    • .btn_xs : height: 36px
    • .btn_xxs : height: 30px
    • .btn_min : height: 24px
    • .btn_ip : height: 50px;

  • [ color 클래스 추가 지정 ]

    • .btn_primary : background, border: #5055b1 / color : #fff
    • .btn_light_gray : background, border: #767676 / color : #fff
    • .btn_line_primary : background: #fff, border: #5055b1 / color : #5055b1
    • .btn_line_gray : background: #fff, border: #ccc / color : #000
    • .btn_line_green : background: #fff, border: #3c9a17 / color : #3c9a17

  • [ shape 클래스 추가 지정 ]

    • .btn_pill : 알약형 버튼
