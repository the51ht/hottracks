Tab 공통규칙
  • jquery UI - Tabs 사용
    - API Url : https://api.jqueryui.com/tabs/
  • 기능 있는 tab 필수 소스
    - 공통 클래스 : .tab_wrap
    - 기본 구조 : .tab_wrap > .tab_list_wrap > ul.tabs > li.tab-item > .tab-link
    - a.tab_link의 href 속성 값과 탭 컨텐츠 영역의 id 값은 동일하게 적용하여 맵핑한다.
      <a href="#탭콘텐츠ID" class="tab_link">
      <div id="탭콘텐츠ID" class="tab_content">
    - 동적으로 해당 tab만 실행필요한 경우 : KyoboBookPub.mok.setTabs('#탭ID값'); 함수 실행.
  • 공통 : 모바일 tab의 경우 가로 해상도 375px이하에서 tab이 넘칠 경우 가로 스크롤 자동 적용
  • 기능 없는 tab 사용 시
    - .tab_wrap에 data-type-btn 추가
  • tab item 활성화 상태 class 제어
    - li.tab_item 영역 .ui-state-active클래스 추가
  • tab item 비활성화 상태 class 제어
    - a.tab_link에 .tab_disabled클래스 추가
  • tab item 비활성화 상태 동적 제어
    - 동적으로 탭 disabled 처리할 경우 $('#탭ID').tabs({disabled: [0, 1]});
    - disabled을 해제할 경우 $('#탭ID').tabs({disabled:[]});과 같이 override 처리
  • 균등형
    - 균등한 크기의 tab을 사용해야할 경우 flex 적용을 위해 .tab_wrap에 .justify 클래스를 추가한다.
  • tab 100% 로 사용해야 하는 경우
    - contents_inner 영역 밖에서 사용하는것을 권장(css에서 -로 사용 지양)
