Tooltip 대체 팝업 (w 295)
  • 모바일에서 tooltip은 팝업으로 대체하여 사용하는 것을 권장한다.
  • 사용법은 기본 레이어 팝업(dialog)와 동일하다.
  • .dialog_wrap에 data-class="dialog_tooltip" 속성 추가 필요
<!-- 팝업 - tooltip 팝업 -->
<div id="popTooltipEx1" class="dialog_wrap" data-class="dialog_tooltip">
	<button type="button" class="btn_dialog_close" data-dialog-close><span class="ico_dialog_close"></span><span class="hidden">닫기</span></button>
	<!-- dialog_header -->
	<div class="dialog_header">
		<div class="dialog_title">tooltip 대체 팝업 타이틀</div>
	</div>
	<!-- //dialog_header -->
	<!-- dialog_contents -->
	<div class="dialog_contents">
		<div class="tooltip_sub_title">리워드 안내</div>
		<div class="tooltip_desc"><a href="#" class="btn_text_link"><span class="text">링크 텍스트</span></a>, 문장수집은 고객님들이 직접 선정한 책의 좋은 문장을 보여주는 교보문고의 새로운 서비스입니다.</div>
	</div>
	<!-- //dialog_contents -->
</div>
<!-- //팝업 - tooltip 팝업 -->
Tooltip
  • [ 툴팁 필수 소스 ]
    - 공통 사용 클래스 : .tooltip_wrap
    - 툴팁 trigger 버튼 : .tooltip_wrap > .btn_tooltip
    - 툴팁 컨텐츠 영역 : .tooltip_wrap > .tooltip_inner
  • 기본 크기 width: 100vw; max-width: 335px; 사이드 여백 20px;
  • 툴팁의 기본 이벤트는 click(toggle)으로 적용된다.
  • [ 툴팁 노출 방향 클래스 ]
    • 추가 클래스 없음(기본) : 왼쪽
    • .right : 오른쪽
  • [ 툴팁 - toggle 타입 ] .btn_tooltip에 속성 추가
    • data-btn-toggle
toggle 타입 : .btn_tooltip에 [data-btn-toggle] 속성 추가
