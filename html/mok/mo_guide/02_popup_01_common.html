Popup 공통규칙
  • jquery UI - Dialog 사용
    - API Url : https://api.jqueryui.com/dialog/
  • script 실행을 위한 dialog 필수 소스
    - popup DOM 구조 추가시, .dialog_wrap 클래스 필요
    - .dialog_wrap에 고유 popup ID 추가 필요 (버튼과 매핑하여 popup 호출 시 사용)
  • dialog on/off
    - open 버튼 : 팝업을 띄울 버튼 태그에 data-role="btn-dialog"와 data-target="#오픈할 팝업 ID"를 필수로 사용한다.
    - close 버튼 : 팝업을 닫을 버튼 태그에 data-dialog-close를 필수로 사용한다.
    - 현재 팝업 close / 새로운 팝업 open 버튼 : 버튼 태그에 data-role="btn-dialog"data-target="#새로 오픈할 팝업 ID"data-dialog-close를 모두 사용한다.
  • 팝업 가이드 타입 추가 필요 시, [data-class]속성에 추가할 클래스 적용한다.
    - [속성없음] : width: 335px; (기본)
    - [data-class="dialog_full"] : full 팝업(width/height: 100%)
    - [data-class="dialog_bottom"] : bottom 팝업
    - [data-class="dialog_alert"] : alert 팝업
  • css overriding이 필요한 팝업일 경우, .dialog_wrap에 멀티클래스 적용한다.
    - .has_btn : 하단 버튼 있는 타입 (버튼 영역 여백 조정을 위한 추가)
    - .no_title : 타이틀 영역 없는 타입 (상단 여백 조정을 위한 추가)
  • 레이어 팝업 필요 시, data-nodim속성을 추가해서 작업하여야 한다
