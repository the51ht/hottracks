Form 공통규칙
  • form 단독 사용 시, input/select/textarea 영역 title값 필수 입력한다.
  • label과 같이 사용되는 form인 경우, title 속성을 생략하고 <label for="">와 <input id="">를 매핑하여 사용한다. (이때, for와 id는 동일한 값을 사용)
  • 비활성화 시, input/select/textarea 태그에 disabled 속성을 추가한다.
  • form 테이블은 table 가이드 참고한다.
input
  • input은 타입별로 구분하여 사용하며 공통 클래스는 .form_ip를 사용
  • password와 search는 기능 사용여부에 따라 가이드 참고
  • [ input 타입 구분 ]
    • 기본, 일반 텍스트 입력 필드 : type="text"
    • 숫자 입력 필드 : type="number"
    • 이메일 입력 필드 : type="email"
    • 전화번호 입력 필드 : type="tel"
    • url 필드 : type="url"
    • password 필드 : input - password (비밀번호 숨김/보임) 가이드 참고
    • search 필드 : input - 검색 (초기화 기능) 가이드 참고
    • ※ 예외 : type="text"에서 숫자 키패드를 노출해야할 경우 inputmode="decimal" pattern="\d*" 속성을 추가하여 사용한다.

  • size 조절은 data-size=""로 제어 (플러그인 기본 기능)
    - md 사이즈 : (22px) default
    - lg 사이즈 : (40px) data-size="lg" (기본 값)
    - sm 사이즈 : (18px) data-size="sm"
    - xs 사이즈 : (13px) data-size="xs"
