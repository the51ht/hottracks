badge 공통
  • 버튼 class는 반드시 size 클래스 1개 + color 클래스 1개의 조합으로 작업한다.

  • [ size 클래스 추가 지정 ] 버튼의 size는 height 기준으로 구분한다.

    • .badge_md : height: 22px
    • .badge_sm : height: 20px

  • [ color 클래스 추가 지정 ]

    • .badge_light_gray : background, border: #767676 / color : #fff
    • .badge_line_primary : background: #fff, border: #5055b1 / color : #5055b1
