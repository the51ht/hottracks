- TYPE2. 추가 상단 고정이 있을 경우
  : [header-wrapper] div 외에 추가 상단이 있을 경우 case

[고정 요소 설명]

  [position: absolute] 상태

  [position: fixed] 상태

공통 지정
- 컨텐츠 높이 변경에 따라 offset 위치 재 지정을 위해 설정 ( 상황에 따라 상단 div거나 부모 div )
   1) 고정 컨텐츠 [sps]에 속성 추가
      : data-height-change="[상위 컨텐츠 클래스명]"
   2) 상위 div 요소에 추가 class
      : [상위 컨텐츠 클래스명] 지정

- 고정 요소 class : [sps sps_scroll_fixed] 지정
- 고정 요소 부모 스타일 추가 : [position: relative], 고정 요소 높이만큼 padding-top 스타일 추가
- .sps > div.sps_inner DOM구조 추가 시, 컨텐츠사이즈(1200px) 기준 가운데정렬

- TYPE2. 추가 상단 고정이 있을 경우
- [sps] 속성 추가 : data-add-offset="[추가 상단 높이 값]"
