파일명세

주석/블럭구분

  • 1탭 들여쓰기 하며, 1탭의 크기는 공백 4칸으로 설정한다.

TODO 주석의 사용

  • TODO 주석을 기입할 경우 타인이 해당 내용을 이해할 수 있도록 상세하게 작성한다.
  • TODO 내용 해결 후 반드시 주석을 삭제한다.

변수(Variables)

  • 코드의 가독성과 유지보수를 위해 변수는 함수 등의 최상단부에 선언(호이스팅, Hoisting)한다.
  • 코드의 통일성을 위해 변수의 기본값을 변수 선언과 같이 하지 않는다.
  • 객체에 속성 추가시 편의성 제고를 위해 객체 리터럴의 ECMAScript5 부터 지원되는 후행 쉼표 처리를 한다.
  • 변수 선언부가 길어지는 경우 개행을 허용하며 개행시 객체 리터럴과 동일한 가독성을 위해 쉼표(,) 이후로 개행한다.

함수(Function)

함수

  • 함수에 설명은 파라미터와 함께 꼼꼼히 작성한다.
  • 작성 규칙은 JSDoc의 CommonJS Modules 규칙 중 함수에 관한 부분을 차용한다.

조건문

  • K&R 방법 : 여는블록({)을 조건과 같은 줄에 배치한다.
  • if / else if와 조건식 사이는 한칸 띄어쓰기 한다

Event

jQuery Event namespace 속성 활용

  • jQuery를 이벤트 등록 후 삭제 또는 트리거 시 네임스페이스를 지정하지 않는 경우 의도치 않은 이벤트를 삭제하거나 이벤트 핸들러가 실행될 수 있다.
  • 이를 해결하기 위한 방법으로 jQuery에서 기본적으로 제공하는 namespace 기능을 활용한다.
jQuery Event namespace 속성정의
네임스페이스명 적용대상 예시
uiCommon 공통 resize, scroll, load $(window).on('resize.uiCommon', function () { ... })
uiGNB setHeader() 함수 내부에서 resize, scroll 이벤트 사용 $(window).off('resize.uiGNB scroll.uiGNB', setScrollUp).on('resize.uiGNB scroll.uiGNB', setScrollUp);
